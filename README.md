# 📐 Quadratic Solver

Welcome to the **Quadratic Solver** project! This tool helps users effortlessly solve quadratic equations of the form ax² + bx + c = 0. 💡

--- 

## 📋 Features
- Solve quadratic equations instantly!  🏎️
- User-friendly interface with clear instructions.
- Visual representation of solutions. ✨

--- 

## 🚀 Installation Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/tagadearpit/Quadratic-solver.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd Quadratic-solver
   ```
3. **Install dependencies** (if applicable):
   ```bash
   npm install
   ```

--- 

## 📖 Usage Guide
1. **Run the application**:
   ```bash
   npm start
   ```
2. **Enter the coefficients** (a, b, c) when prompted.
3. **View the results!** 🥳
